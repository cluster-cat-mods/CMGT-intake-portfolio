Inleiding

Voor deze opdracht kregen wij de opdracht om een module te kiezen, deze modules stonden in een lijst maar men mocht ook andere modules kiezen, en de opdrachten van deze modules te maken, om zo uit te komen op één eindopdracht. Wij hebben voor deze opdracht een module over Unity op het internet gevonden en hebben besloten om hiermee te werken omdat het maken van games en het werken met Unity ons interesseert

 

Opdrachtomschrijving

We hebben gekozen om de module ‘een game in Unity maken’ te doen. Hierbij hebben we ervoor gekozen om de module niet strikt te volgen, aangezien de module best groot is en we ons liever willen focussen op de game zelf in unity maken. We hebben vooral de opdrachten van het maken van een concept gemaakt en verschillende tutorials in de module gevolgd. Als eindproduct hebben we bedacht om een 2D top-down survival game te maken waarbij je verschillende items en wapens kan verzamelen en bepaalde spells en magie kan gebruiken. Het doel in de game is om het zo lang mogelijk vol te houden en zo veel mogelijk wapens, armor en spells te verzamelen. Een voorbeeld van en spell die we willen toevoegen is een bliksemschicht die vanuit de lucht op de enemies neerkomt en AOE damage doet. Als enemies hadden we in gedachten om verschillende vierkante monsters te maken, zoals een steenkubus.

 

Projectmanagement

We gaan aan het begin van elke les inventariseren wat er gedaan moet worden en wie wat gaat doen. We noteren elke les kort wat we hebben besproken in een lijst bij de planning in Trello. Degene die dit doet, zet zijn naam erbij.

- omgaan met bestanden / backups.
Iedereen plaatst hun backups en bestanden in hun eigen backup map in de gedeelde Google Drive.

Wat we tot nu toe hebben gedaan

We zijn begonnen met het bedenken van een concept, hierbij hebben we het eerste deel van de opdrachten in de module gemaakt. Daarna hebben we meerdere tutorials uit de module gevolgd waaronder de Roll-a-ball tutorial. Daarna zijn we begonnen met het maken van de game zelf. We hebben nu het grootste deel van de map af, voor de textures van de map hebben we een template uit de Unity asset store gebruikt. Voor de map zelf hebben we Unity’s tilemap systeem gebruikt met prefab assets voor objecten zoals bomen en planten, waarvan we sommige textures nog willen aanpassen. Verder hebben we een player die kan lopen en health heeft, we hebben ook al een texture voor deze player, maar we zijn van plan om de speler te laten kiezen uit meerdere skins. Ook hebben we al een werkend script voor de enemies en en texture voor een van de enemies. En als laatste zijn we begonnen aan de UI en aan het combat systeem.

Eindopdracht

Als eindopdracht hebben wij besloten om een 2D top-down dungeon crawler te maken, waarin de speler als een kubuskat zich door een leven beweegt en andere kubus vijanden moet verslaan.

De character als wie de speler speelt is Cubic Carfield. Deze kubusvormige kat kan met zijn vuurbal de verschillende kubus enemies verslaan. Deze enemies, bijvoorbeeld de grumpy crate en de wonky wall, achtervolgen de player door de map om te proberen de player aan te vallen als deze dichtbij genoeg is. De player heeft de mogelijkheid om verschillende object die verspreid zijn over de map te verplaatsen om zo de enemies te hinderen, de enemies kunnen deze objecten echter ook bewegen.

De player heeft 10 health en raakt elke keer als hij geraakt wordt door een enemy 2 health kwijt, de bedoeling is dat de player alle enemies in het level verslaat om het level te winnen.

 

We hebben voor zowel de enemies als de player een script gemaakt voor health. Elke keer als een enemy een player aanraakt wordt er 2 health bij de player verwijderd, als de player 0 of minder health heeft wordt de player vernietigd. Hetzelfde geld voor de enemies, die hebben alleen 3 health in plaats van 10 en krijgen geen damage als ze door de player worden geraakt.

Ook hebben we een script gemaakt om de enemy de player te laten volgen, het enige wat we niet voor elkaar konden krijgen is om de enemy op dezelfde manier te laten lopen als de player. Als de enemy nu draait, draait deze de texture mee, in plaats van alleen 90 graden bochten te maken met een texture voor elke richting. Dit zorgt ervoor dat de enemy niet altijd recht achter de player staat en dat de vuurbal moeilijk is om te raken.

using System.Collections;

using System.Collections.Generic;

using UnityEngine;

using UnityEngine.InputSystem;

 

public class playershooot : MonoBehaviour

{

    [SerializeField]

    public GameObject ProjectilePrefab;

    [SerializeField]

    private float Projectilespeed;

    [SerializeField]

    private float timebetweenshots;

    private bool FireContinuously;

    private float lastfiretime;

 

    // Update is called once per frame

       void Update()

    {

        if (Input.GetKeyDown(KeyCode.Space))

        {

            Debug.Log("Space key was pressed.");

            float timesincelastfire = Time.time - lastfiretime;

            if (timesincelastfire >= timebetweenshots)

            {

                FireProjectile();

                lastfiretime = Time.time;

            }

          

        }

      

    }

  

    private void FireProjectile()

    {

        GameObject Projectile = Instantiate(ProjectilePrefab, transform.position, transform.rotation);

        Rigidbody2D rigidbody = Projectile.GetComponent<Rigidbody2D>();

        rigidbody.velocity = Projectilespeed * transform.up;

    }

 

    private void OnFire(InputValue inputValue)

    {

        FireContinuously = inputValue.isPressed;

 

    }

}

using System.Collections;

using System.Collections.Generic;

using UnityEngine;

 

public class enemy : MonoBehaviour

{

    public GameObject player;

    public float speed;

    public float distancebetween;

 

    private float distance;

 

    void Start()

    {

        // player = GameObject.FindGameObjectWithTag("Player");

 

    }

 

    void Update ()

    {

      

        distance = Vector2.Distance(transform.position, player.transform.position);

        Vector2 direction = player.transform.position - transform.position;

        direction.Normalize();

        float angle = Mathf.Atan2(direction.y, direction.x) * Mathf.Rad2Deg;

 

        if(distance < distancebetween)

        {

          

        transform.position = Vector2.MoveTowards(this.transform.position, player.transform.position, speed * Time.deltaTime);

        transform.rotation = Quaternion.Euler(Vector3.forward * angle);

 

        }

    }

}
